---
name: extract-chat
description: 用于将当前对话内容精简成一份方案或设计文档，以便其他agent接手。
---

总结当前对话内容，不要遗漏，不要改变了意图，编写一份方案或设计文档，以便下一位agent能够继续工作。将文档保存到`docs/yo-analy/YYYY-MM-DD-<topic>-chat.md`的临时目录。

不要重复其他文档（产品需求文档、计划、应用需求文档、问题、提交记录、差异）中已包含的内容。请使用路径或 URL 引用它们。

删除任何敏感信息，例如 API 密钥、密码或个人身份信息。

如果用户传递了参数，请将其视为对下次会话重点内容的描述，并据此调整文档。
