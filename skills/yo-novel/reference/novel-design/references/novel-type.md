# 小说类型指导

不同类型的小说有不同的设计侧重点。在开始设计前，先明确类型，再围绕该类型的核心诉求展开。

## 玄幻/仙侠

- **重点**：力量体系、升级路径、世界等级、势力格局。
- **注意**：规则要清晰，战力不要崩坏；世界为爽点和主角成长服务。
- **常见结构**：废柴崛起 → 获得机缘 → 不断突破 → 登顶/揭示更大世界。

## 奇幻

- **重点**：魔法系统、种族与文化、神话背景、独特生态。
- **注意**：魔法规则要有限制和代价；文化细节让世界可信。
- **常见结构**：平凡者进入异世界 → 组建队伍 → 对抗黑暗势力。

## 科幻

- **重点**：科技设定、未来社会结构、科学逻辑的合理性。
- **注意**：硬科幻要重视科学自洽；软科幻可以借科技外壳探讨人性。
- **常见结构**：发现异常 → 探索未知 → 科技/伦理冲突 → 新的认知。

## 悬疑/推理

- **重点**：核心谜题、线索布局、误导与反转、侦探/反派智商在线。
- **注意**：伏笔要公平，解答要令人信服；节奏比世界观更重要。
- **常见结构**：案件发生 → 调查推理 → 连续反转 → 真相揭露。

## 言情

- **重点**：人物关系、情感张力、角色成长、阻碍与磨合。
- **注意**：感情线要自然，避免为虐而虐；人物动机要真实可信。
- **常见结构**：相遇 → 吸引 → 阻碍 → 突破 → 确定关系。

## 都市/现实

- **重点**：社会背景、职业细节、人物困境、真实感。
- **注意**：贴近生活经验，避免悬浮；冲突来自现实矛盾。
- **常见结构**：主角面临现实压力 → 挣扎与选择 → 成长或和解。

## 历史/架空历史

- **重点**：时代背景、政治格局、社会风俗、历史事件改编。
- **注意**：尊重历史逻辑；架空部分要与真实历史融洽。
- **常见结构**：小人物卷入大时代 → 见证历史 → 改变命运。

## 恐怖/惊悚

- **重点**：恐惧来源、氛围营造、未知与不可名状、心理压迫。
- **注意**：恐惧要具体且持续；过度解释会削弱恐怖感。
- **常见结构**：异常出现 → 调查真相 → 规则逐渐清晰 → 生死对抗。

## 通用提醒

- 不要一次性设定完所有细节，先抓核心。
- 世界和人物要为故事服务，避免过度构建。
- 留下可展开的空间，给后续创作和读者想象留有余地。
