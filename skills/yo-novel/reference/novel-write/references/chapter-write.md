## 情绪曲线

用起承转合规划本章节奏：

- **起**：开篇引入，交代背景，建立预期
- **承**：发展情节，层层推进，加深冲突
- **转**：转折变化，出人意料，情绪达到高点
- **合**：收束本章，点明主题，留下钩子

## 写作检查清单

写完后逐项检查：

- [ ] 字数在 2000-4000 之间
- [ ] 没有出现占位符或未完成段落
- [ ] 叙事视角在本章内保持一致
- [ ] 每个重要场景都有时间、地点、人物、冲突、变化
- [ ] 对话推动情节或展现性格，而非单纯交代信息
- [ ] 动作、表情、细节描写足够，避免大量直接陈述
- [ ] 高潮与缓冲交替，节奏张弛有度
- [ ] 逻辑自洽，没有违背已有设定
- [ ] 伏笔埋设或回收自然，能服务后续情节
- [ ] 与上一章结尾衔接自然
- [ ] 本章末尾有情绪落点或下一章钩子
