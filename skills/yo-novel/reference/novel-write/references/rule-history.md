# 历史类小说规则参考

适用于以真实或架空历史为背景的小说。

## 基本准则

1. **尊重基本史实（或给出明确架空声明）**
   - 若使用真实历史背景，重大事件、制度、人物关系应符合史实，或明确说明是架空/戏说。
   - 若架空，需建立清晰的世界规则并保持一致。

2. **时代语言与称谓**
   - 对话与叙述应避免明显超越时代的词汇和观念。
   - 称谓、礼仪、官职、地名应符合时代背景。

3. **社会与生活细节**
   - 服饰、饮食、建筑、交通、器物等细节要符合时代特征。
   - 不要把现代生活方式简单套用到古代。

4. **人物动机与价值观**
   - 古代人物的思维方式、价值判断与现代人不同，应避免全员现代思维。
   - 主角可以有现代视角，但需通过合理方式呈现（如穿越设定）。

## 常见雷区

- 现代政治观念、法律术语直接套用到古代。
- 重要历史人物行为严重 OOC（脱离其历史形象）而没有合理铺垫。
- 武器、科技、经济体系超越时代却没有解释。
- 把"历史"写成"披着古装的现代剧"。

## 写作建议

- 历史是背景，故事是核心；不要为了展示史实而牺牲叙事节奏。
- 可以在史实缝隙中合理想象，但要标注哪些是史实、哪些是虚构。
- 多查资料，对不确定的细节保持谨慎。
