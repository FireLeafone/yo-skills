# 古风写作风格

适用于古代背景、仙侠、武侠、历史演义等题材。

## 语言基调

- 用词偏书面化、典雅化，避免现代口语和网络用语。
- 可适当使用四字词语、成语、典故，但不过度堆砌。
- 句式长短错落，讲究音韵与节奏。

## 叙事特点

- 场景描写注重意境，常用虚实结合的手法。
- 人物动作带有古典礼仪感或江湖气。
- 对话可夹杂文言虚词（如"汝"、"吾"、"尔"、"岂"、"乎"、"也"），但要把握分寸，避免全文晦涩。

## 常用表达

- 时间：子时、卯时、黄昏、月上柳梢
- 地点：客栈、酒楼、山门、渡口、长安街、竹林深处
- 动作：拱手、抱拳、拂袖、策马、提剑、敛衽
- 情绪：眸色一沉、眉心微蹙、神色晦暗、嘴角噙笑

## 注意事项

- 避免现代词汇突兀出现（如"手机"、"公司"、"OK"）。
- 不同朝代的社会风俗、称谓、器物要符合基本史实或架空设定。
- 打斗场面要有画面感，但不宜过度血腥直白。
