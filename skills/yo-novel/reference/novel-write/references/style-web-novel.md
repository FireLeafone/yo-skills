# 网文写作风格

适用于网络小说、爽文、玄幻、都市异能、游戏异界等追求阅读快感的题材。

## 语言基调

- 简洁直白，一句话一个信息点。
- 节奏快，尽量少用长段落和复杂修辞。
- 情绪外露，爽点、笑点、怒点要清晰可感。

## 叙事特点

- 开篇尽快进入冲突或悬念，减少铺垫。
- 每章结尾尽量留钩子，吸引读者继续下一章。
- 打脸、反转、升级、收获等爽点要足，铺垫要充分。
- 配角反应常用来衬托主角，形成"震惊流"效果。

## 节奏控制

- 铺垫 → 冲突 → 爆发 → 收获 → 反响，循环推进。
- 一个小高潮后，用简短过渡进入下一个事件。
- 避免连续大段描写，保持段落简短，便于手机阅读。

## 常用技巧

- **打脸**：反派嚣张 → 主角隐忍或反击 → 反派震惊/后悔。
- **升级**：明确展示主角能力、地位、财富的提升。
- **收获**：获得宝物、技能、人脉、信息等优势。
- **震惊流**：通过配角、路人的反应放大主角行为的冲击力。

## 注意事项

- 爽点要有逻辑，不能为爽而爽导致设定崩坏。
- 反派不宜过于弱智，否则胜利缺乏成就感。
- 保持主角目标清晰，每章都向目标推进。
- 避免水字数，每个场景都服务于情节或人物。
