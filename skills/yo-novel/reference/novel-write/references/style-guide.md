# 风格指南

本文件补充 `SKILL.md` 中提到的内置写作风格。仅在用户未指定风格、需要展开说明时读取。

## 内置风格

1. **幽默风趣**
   - 轻松诙谐，善用反讽、意外和自嘲。
   - 适合喜剧、轻小说、日常流。

2. **犀利紧凑**
   - 短句密集，节奏快，冲突强。
   - 适合悬疑、动作、快节奏类型。

3. **叙事陈述**
   - 平稳推进，重情节与信息交代。
   - 适合史诗、纪实感强或信息密度高的场景。

4. **感人煽情**
   - 重情绪渲染，细腻刻画人物内心。
   - 适合情感戏、离别、牺牲、成长节点。

5. **场面宏大**
   - 多线交织，气势恢宏，适合群像与大场景。
   - 适合战争、仙侠大战、历史转折。

6. **自然人声**
   - 口语化表达，贴近真实对话与思维。
   - 参考 `references/style-natural-voice.md`。

7. **古风文言**
   - 古典韵味，书面化、典雅化。
   - 参考 `references/style-ancient.md`。

8. **网文爽感**
   - 节奏快、爽点密集、情绪外露。
   - 参考 `references/style-web-novel.md`。

## 题材特殊参考

- 历史类小说额外参考 `references/rule-history.md`。
- 如果用户指定了项目自定义风格或提供了例文，优先使用用户提供的风格，本文件仅作为兜底参考。
